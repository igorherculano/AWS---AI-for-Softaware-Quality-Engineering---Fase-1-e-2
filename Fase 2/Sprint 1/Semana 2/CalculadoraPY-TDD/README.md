# README - Testes de Software
Este projeto implementa uma calculadora com foco em testa-la através de frameworks do Python.

## Tecnologias
Python 3.x

Pytest: Framework de testes

## Estrutura de Testes
Soma: Validação de inteiros, negativos e floats

Subtração: Testes de diferença e operações com zero

Multiplicação: Regra de sinais e fatores decimais

Divisão: Cobertura de quocientes e dízimas periódicas

Casos de Borda (Edge Cases)

ZeroDivisionError: Validação de erro ao dividir por zero

TypeError: Garantia de que apenas entradas numéricas são processadas

## Execução
Instale o Pytest: pip install pytest

Execute: pytest test_calculadora.py (Dentro de alguma IDE)
