# 📂 Repositório de Atividades e Artefatos - PB

Bem-vindo ao repositório central de atividades. Este projeto tem como objetivo versionar e armazenar todos os artefatos produzidos durante as Sprints anteriores, consolidando o aprendizado prático em **Quality Assurance (QA)** e **Testes de Software**.

Aqui você encontrará documentações, planos de testes, mapas mentais e coleções de API, organizados para facilitar a consulta e a evolução contínua dos estudos.
